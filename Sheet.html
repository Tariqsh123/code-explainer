Tag / Element Purpose (Easy Words)
<!DOCTYPE html>Ye browser ko batata hai ke ye HTML5 document hai.
<html>Poore page ka main container.
<head>Page ki hidden info (title, CSS, scripts).
<title>Page ka naam jo browser tab me dikhta hai.
<body>Page ka asli content jo user ko dikhega.
<meta>Hidden info jaise charset, keywords, viewport.
<link>CSS file ya icons link karne ke liye.
<style>CSS likhne ke liye (andar hi).
<script>JavaScript likhne ya add karne ke liye.
<noscript>Message dikhane ke liye agar JS off ho.
<base>Page ke sab links ke liye base URL set karta hai.
<h1> to <h6>Heading levels (bada se chhota).
<p>Normal paragraph likhne ke liye.
<br>Line todne ke liye (next line).
<hr>Page me horizontal line kheenchne ke liye.
<pre>Text ko jaisa likha hai waisa hi dikhata hai (spaces, lines).
<blockquote>Bade quotation (quote) dikhane ke liye.
<q>Chhota quotation dikhane ke liye.
<abbr>Short form dikhane ke liye (tooltip milta hai).
<cite>Kisi source ka naam ya reference.
<code>Computer code dikhane ke liye.
<kbd>Keyboard input dikhane ke liye (jaise Ctrl+C).
<samp>Computer ka output dikhata hai.
<var>Variable ya math formula ka naam.
<b>Bold text (sirf style).
<strong>Important text (bold bhi).
<i>Italic text (sirf style).
<em>Emphasized text (italic bhi).
<mark>Text ko highlight karne ke liye.
<small>Chhota text banane ke liye.
<del>Delete ya cut hua text (line kheecha hoga).
<ins>Naya add hua text (underline).
<sub>Neeche wala chhota text (H₂O).
<sup>Upar wala chhota text (x²).
<span>Inline container (sirf styling ke liye).
<a>Dusre page ya link ke liye.
<nav>Navigation menu (links ka group).
<ul>Bullet points list (unordered).
<ol>Number wali list (ordered).
<li>List ke item.
<dl>Description list (jaise glossary).
<dt>Term ya word.
<dd>Us word ki explanation.
<img>Image dikhane ke liye.
<audio>Audio player lagane ke liye.
<video>Video player lagane ke liye.
<source>Audio/video ki file source.
<track>Subtitles ya captions video ke liye.
<picture>Alag-alag size ki image dikhane ke liye.
<map>Image ke andar clickable map.
<area>Image ke andar clickable area.
<canvas>Drawing ya graphics JS se banane ke liye.
<svg>Vector graphics banane ke liye.
<iframe>Dusra page embed karne ke liye.
<embed>External cheez (PDF, video) embed karne ke liye.
<object>External file embed karne ke liye (old).
<param>Object ke liye setting define karne ke liye.
<form>Form banane ke liye (inputs ke liye).
<input>Text box, checkbox, etc. ke liye.
<textarea>Bada text box.
<button>Button banane ke liye.
<select>Dropdown menu.
<option>Dropdown ka option.
<optgroup>Dropdown me group banane ke liye.
<label>Input ke liye label (naam).
<fieldset>Form ke inputs ko group karne ke liye.
<legend>Fieldset ka title.
<datalist>Input ke suggestions dene ke liye.
<output>Form ka result dikhane ke liye.
<progress>Progress bar.
<meter>Measurement bar (jaise battery level).
<table>Table banane ke liye.
<tr>Table row.
<td>Table cell (data).
<th>Table header cell.
<thead>Table ka header part.
<tbody>Table ka main body part.
<tfoot>Table ka footer part.
<caption>Table ka title.
<col>Table column ke style ke liye.
<colgroup>Columns ka group banane ke liye.
<header>Page ka top section.
<footer>Page ka bottom section.
<main>Page ka main content.
<section>Page ka ek section.
<article>Independent content (blog post).
<aside>Side ka content (sidebar, ads).
<address>Address ya contact info.
<div>Content ka container (block).
<details>Extra info jo open/close ho sakti hai.
<summary>Details ka title.
<dialog>Popup box ya modal.
<template>Hidden content jo later use hoga.
<slot>Web components me placeholder.
<font>Text ka font set karta tha (ab CSS se karte hain).
<center>Content center me lata tha (ab CSS se karte hain).
<big>Text bada karta tha (ab CSS use karo).
<marquee>Scrolling text banata tha (fun tha, ab obsolete).
<applet>Java applets ke liye (ab band).
<acronym>Acronym ke liye (ab <abbr> use karo).
<tt>Teletype font dikhata tha (ab CSS use karo).
<frameset>Page ko frames me todta tha (ab nahi use hota).
<frame>Frameset ka ek frame (ab obsolete).
<noframes>Frames ko support na karne wale browsers ke liye.
<bgsound>Background me music bajata tha (sirf IE me).
<blink>Text blink karta tha (sirf purane Netscape me).
<keygen>Keys generate karne ke liye (ab remove ho gaya).
<shadow>Shadow DOM ke liye (ab remove ho gaya).

Keyword / Function Purpose (Easy Words)
let / const / varVariable declare karne ke liye (let/const block ke andar, var function ke andar valid).
function / arrow function (=>)Reusable code block; arrow function short syntax: const f = () => {}
returnFunction se value bahar bhejne ke liye.
for / while / do...whileLoops: for fix number of times, while tab tak jab condition true, do...while pehle ek baar fir check.
if / else / else if / switch / case / defaultConditional statements aur multiple options ke liye switch.
break / continueLoop ya switch se bahar nikalne ya skip karne ke liye.
array / [] / object / {}Multiple values store karne ke liye array ya key-value pairs ke liye object.
.push() / .pop() / .shift() / .unshift()Array me item add/remove karne ke liye.
.lengthArray ya string me items ki ginti.
.indexOf()Array me value ka index find karta hai ya -1 agar nahi mile.
string / .toUpperCase() / .toLowerCase() / .slice() / .replace() / .includes()Text store aur manipulate karne ke liye.
document.getElementById() / querySelector() / querySelectorAll()HTML elements select karne ke liye.
.innerHTML / .textContent / .style / .addEventListener()Element ke content, style change aur events handle karne ke liye.
console.log()Browser console me message print karne ke liye.
alert()Popup message dikhata hai.
prompt()User se input lene ke liye popup.
parseInt() / parseFloat()Text ko number me convert karne ke liye.
Math.round() / Math.floor() / Math.ceil() / Math.random()Number calculations aur random number generate karne ke liye.
+ / - / * / / / % / == / === / != / !== / > / < / >= / <= / && / || / !Operators: math, comparison aur logical operations.
typeof / new / this / null / undefined / NaNType check, new object, current object, aur special values.
